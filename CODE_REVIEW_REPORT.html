1. Feature-off WebSocket 仍可透過 response header 協商 otel-ws

otel-gorilla-ws/upgrader.go:121-126

當 tracing feature 關閉且 appProtocols 為 nil，原始 responseHeader 仍會傳給 gorilla upgrader。header 內的 otel-ws／otel-ws+json 可因此被確認，但回傳的 Conn 保持 tracingEnabled=false。
